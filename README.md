# 제2회 유통데이터 활용 경진대회


## 프로젝트 소개
전라북도 대형물류 유통 데이터를 이용한 상품 판매량 예측과 아이디어 공모


## 개발 기간
 - 2023.8.14 ~ 2023.9.1


### 멤버 구성
 - 강새롬(AA): 총괄, 알고리즘 설계 및 구현
 - 전유진(TA): 데이터 전처리, 데이터 라벨링
 - 조하연(TA): 알고리즘 설계 및 구현
 - 전소라(DA): PPT 제작
 - 이하영(DA): 데이터 시각화


### 개발 환경
 - 운영 체제: Windows 10
 - 개발 언어: Python 3.8
 - 개발 도구: VS Code
 - 분석 및 시각화: Power BI, matplotlib, Seaborn, Pandas
